# CoinMarketApp
React native app that lists all the cryptocurrencies from Coinmarketcap. Supports both platforms iOS/Android. 

## Features
* Uses CoinMarketCap api V2 (https://coinmarketcap.com/api/)
* Axus for http requests (https://github.com/axios/axios)
* React navigation (https://reactnavigation.org/)
* Redux for state managment (https://redux.js.org/)
* Realm for local storage (https://realm.io/docs/javascript/latest/api/index.html)
* Promises for asynchronous operations
